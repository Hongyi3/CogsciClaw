# Code of conduct

We want this project to be a respectful, rigorous, and welcoming place for collaboration.

## Expectations

Participants in this project are expected to:

- be respectful in disagreement
- critique ideas, methods, and code without attacking people
- assume good faith while still holding work to a high scientific standard
- avoid harassment, discrimination, and exclusionary behavior
- protect confidentiality when sensitive research contexts are involved

## Scope

This code of conduct applies to repository spaces, issue discussions, pull requests, review comments, and project-related public interactions.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers privately.

## Enforcement

Maintainers may remove content, moderate discussions, or restrict participation when necessary to protect contributors and the project.

## Note

Before public launch, replace or align this with your chosen formal open-source code of conduct policy (for example, Contributor Covenant 3.0).
