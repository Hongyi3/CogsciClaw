# Benchmarks

This directory defines benchmark intent and registry metadata.

## Benchmark principles

- use public or explicitly shareable data only
- freeze dataset versions for public benchmark claims
- distinguish demos from benchmarks
- keep benchmark commands and expected outputs versioned

## Minimum benchmark pool

- behavioral Flanker demo
- questionnaire-heavy example
- EEG oddball example
- one public BIDS example
- one public Psych-DS example
- later: one DANDI / NWB example
