# Vision

## Mission

Create the most credible open-source platform for **AI-assisted, standards-compliant, reproducible cognitive-science workflows**.

## Primary users

- cognitive science labs
- graduate students and postdocs
- methods-focused researchers
- open-science infrastructure builders
- research software engineers

## Core promise

A user should be able to move from a well-formed study request or study specification to:

- runnable task materials
- standards-compliant data structures
- validated metadata
- reproducible analysis
- manuscript-grade outputs
- preregistration-ready materials
- provenance and citation metadata

## What makes the project academically serious

- it wraps community standards instead of bypassing them
- it emits validator outputs instead of vague “compliance” claims
- it separates confirmatory and exploratory analyses
- it generates methods text from actual run metadata
- it treats provenance as part of the output, not an afterthought

## V1 scope

### In scope

- behavioral task generation
- questionnaire / assessment structure
- Psych-DS curation
- BIDS-aligned intake for EEG/MEG
- HED event annotation
- Bayesian and drift-diffusion modeling
- report + provenance + preregistration bundle
- agent-facing repository infrastructure for Codex

### Out of scope for v1

- public skill marketplace
- unsupported clinical claims
- unsupervised interpretation of participant-level results
- custom replacement for fMRIPrep
- full neurophysiology archive workflow
- generalized “AI psychologist” features

## Success criteria

### Scientific

- external researchers can reproduce demos without contacting maintainers
- outputs are standards-aware, validated, and citable
- benchmark claims are preregistered and auditable

### Software

- each supported skill has a demo, tests, and generated docs
- public capability listings do not drift from the source catalog
- CI enforces catalog/document synchronization

### Community

- contributors can add new skills with low friction
- maintainers can review contributions against stable criteria
- labs can adopt the project without trusting opaque AI behavior
