# Security policy

## Reporting a vulnerability

Please report suspected security issues privately to the maintainers before opening a public issue.

Include:

- affected files or skill paths
- reproduction steps
- impact
- any logs or screenshots with sensitive information redacted

## Scope

Security concerns include, but are not limited to:

- unsafe skill execution
- prompt injection vectors
- sensitive data leakage
- dangerous default permissions
- supply-chain risks
- insecure example configurations

## Supported versions

Until the first stable release, security fixes will target the default branch.
