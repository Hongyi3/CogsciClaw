# ADR-0001: standards-first v1 scope

## Status

Accepted

## Context

The project could launch either as a broad “AI for cognitive science” assistant or as a narrower standards-first workflow system.

## Decision

Launch as a standards-first workflow system with a narrow v1 wedge:

- behavioral tasks
- behavioral data curation
- HED event annotation
- Bayesian and DDM analysis
- reproducibility bundle
- EEG/MEG expansion after the behavioral vertical slice works

## Consequences

### Positive

- higher credibility
- easier benchmarking
- better external adoption pathway
- clearer release gates

### Negative

- slower apparent breadth
- some modalities deferred

## Follow-up

Revisit scope after the first behavioral and EEG/MEG vertical slices are benchmarked.
