# License decision

## Recommendation

Use **Apache-2.0** for the core repository unless you have a strong institutional reason to choose otherwise.

## Why Apache-2.0 is the default recommendation

- permissive for academic and industry collaboration
- explicit patent grant
- familiar to research-software ecosystems

## Additional note

Code license does **not** automatically cover:

- questionnaires
- copyrighted stimuli
- proprietary task assets
- third-party datasets
- benchmark data redistribution rights

Audit those separately before public release.

## Action required before launch

Add a real `LICENSE` file before the first public release.
